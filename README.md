# Factor Model

## 1. Install

  * *Windows*

    * Added ``lib`` folder into ``PATH``;

    * Typed following command:
      ```
      python setup.py install
      ```


  * *Linux*

    * Added ``lib`` folder into ``LD_LIBRARY_PATH``;

    * Typed following command:
      ```
      python setup.py install
      ```
